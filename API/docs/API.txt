*** User ***

GET /user/get/<user_id>

    Response:
    {name: name,
     surname: surname,
     birth_date: birth_date,
     bio: bio,
     hometown: hometown,
     registration_date: registration_date,
     event: event_id}


PATCH /user/update/<user_id>

    Request:
    {name: name,
     surname: surname,
     birth_date: birth_date,
     bio: bio,
     hometown: hometown}


DELETE /user/delete/<user_id>

    Request:
    {name: name,
     login_email: login_email,

     age: age,
     hometown: hometown}

    Request:
    {user_id: user_id,
     password: password}

    Response:
    {refresh_token: refresh_token,
     access_token: access_token}
